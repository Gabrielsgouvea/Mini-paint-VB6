# Mini-paint-VB6
VB6 Paint — Estudos de 2012
Um editor de imagens simples feito em Visual Basic 6, desenvolvido como exercício de aprendizado para explorar recursos da linguagem e da API do Windows.
Contexto
Projeto criado em 2012 durante estudos de VB6. O objetivo não era construir um software completo, mas testar funcionalidades específicas da linguagem e entender como interagir com eventos do sistema operacional.

Funcionalidades implementadas

Desenho livre com lápis, usando controle de espessura da linha
Caixa de cores com paleta de cores predefinidas e seletor de cor atual
Salvar / Abrir imagem — leitura e gravação de arquivos de imagem no disco
Drag and drop — arrastar e soltar para interação com elementos da interface
Conta-gotas via scroll do mouse — ao pressionar o botão do meio (scroll), o cursor entra em modo conta-gotas, capturando a cor do pixel sob o ponteiro e aplicando-a como cor ativa do lápis
Múltiplas janelas de canvas com suporte a troca entre elas
Borracha com cor configurável

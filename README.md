# Mini-paint-VB6
VB6 Paint — Estudos de 2012

Um editor de imagens simples feito em Visual Basic 6, desenvolvido como exercício de aprendizado para explorar recursos da linguagem e da API do Windows.

## Contexto
Projeto criado em 2012 durante estudos de VB6. O objetivo não era construir um software completo, mas testar funcionalidades específicas da linguagem e entender como interagir com eventos do sistema operacional.

## Funcionalidades implementadas

- Desenho livre com lápis, com controle de espessura da linha
- Caixa de cores com paleta predefinida e seletor de cor atual
- Salvar/Abrir imagens — leitura e gravação de arquivos no disco
- Drag and drop — arrastar e soltar para interação com elementos da interface
- Conta-gotas via scroll do mouse — ao pressionar o botão do meio, o cursor entra em modo conta-gotas, capturando a cor do pixel sob o ponteiro e aplicando-a como cor ativa do lápis. Funcionava melhor no Windows XP; no Windows 11 há uma diferença entre a posição do cursor e a cor capturada
- Múltiplas janelas de canvas, com suporte à alternância entre elas
- Borracha com cor configurável (botão direito do mouse)
